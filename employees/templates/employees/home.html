REST API v.1.0



Employee

Resource Description Example Payload
GET /api/v1/employee/ Retrieve all the registered employees.
POST /api/v1/employee/ Create a new employee. '{"name":"Employee Name", "email": "some@email.com", "department": "Business"}'
GET /api/v1/employee/:id Receive a single employee.
PUT /api/v1/employee/:id Modify attribute/attributes for an existing employee. '{"department": "Developer"}'
DELETE /api/v1/employee/:id Remove a employee from the database.
